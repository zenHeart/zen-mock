## 项目说明
一套简化 API mock 流程的工具链
详细功能参考工具文档.

* [zen-mock](./packages/zen-mock/README.md) 基于配置快速模拟 json 服务器
* [zen-mock-cli](./packages/zen-mock-cli/README.md) 利用 mock 配置校验实际服务器数据
* [zen-api-parser](./packages/zen-api-parser/README.md) API 配置文件解析工具
